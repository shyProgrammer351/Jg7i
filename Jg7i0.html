ServerSocket   서버에서 네트워크를 개설하기 위한 소켓

Socket   특정 IP주소와 포트번호의 서버에 클라이언트가 연결 요청하면 서버는 수락하고 서버-클라이언트를 1:1로 연결하기 위한 네트워크 소켓

accept   클라이언트에서 서버로 연결 요청시 이를 받아들여 연결 활성화하며, public Socket accept()에서 반환값은 클라이언트와 1:1로 연결된 소켓
 

멀티스레드 기반 클라이언트-서버 구조

단일스레드 기반클라이언트-서버 구조에서는 데이터를 보내고 받는 순서를 반드시 지켜야 한다는 제한 발생
클라이언트는 데이터를 보낸 후 데이터를 받을 수 있고, 서버는 데이터를 받은 후 데이터를 보낼 수 있음
네트워크 데이터를 받아서 처리하는 전용 스레드를 사용하면, 클라이언트와 서버는 데이터 보내기만 집중 가능
